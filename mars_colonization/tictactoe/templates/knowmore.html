Features of the project

ALGORTITHMS

Apart from Minimax Algorithm, we have also implemented titctactoe AI using Neural Networks and Logic Based Algorithm to give it a unique touch

DIFFICULTY LEVELS

There are 3 difficulty levels for each AI algorithm implemented- Easy, Medium and Unbeatable.

FUNCTIONAL PARADIGM

Agent behavior has been emulated using Functional Programming paradigm with functions that act on other functions.

REVERSE TICTACTOE

Reverse TicTacToe ,another unique feature of our project, has been implemented using Minimax algortihm.

HINTS

An option for hint,another unique feature, is provided which suggests the best possible next move to Human player.

PvP WITH 2 GRIDS

Player vs Player option with 3x3 and 4x4 grid sizes is present for multi-player gaming making it yet another unique feature of our app.
